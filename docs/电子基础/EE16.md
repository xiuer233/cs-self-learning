# EE16A&B: Designing Information Devices and Systems I&II
## 课程简介
- 所属大学：UC Berkeley
- 先修要求：无
- 编程语言：Python
- 课程难度：🌟🌟🌟
- 预计学时：150小时

UC Berkeley电子系学生的大一入门课，通过电路基础知识的讲授，配合各类动手实操的lab，让学生体验通过电路从环境中收集信息并进行分析，作出预测和反馈。由于疫情的缘故，所有lab都有远程在线版，非常适合大家在家自学。
